Minecraft

Minecraft is an open-world sandbox game developed by Mojang Studios. It allows players to explore infinite worlds, build structures, gather resources, and survive against various dangers.

Key Information

  • Developer: Mojang Studios
  • Release Date: November 18, 2011
  • Platforms: PC, PlayStation, Xbox, Nintendo Switch, Mobile
  • Game Modes: Survival, Creative, Adventure, Spectator

Notable Characters

  • Steve: The default male player character.
  • Alex: The default female player character.
  • Creeper: A hostile mob that explodes when near the player.
  • Ender Dragon: The final boss in the game.
