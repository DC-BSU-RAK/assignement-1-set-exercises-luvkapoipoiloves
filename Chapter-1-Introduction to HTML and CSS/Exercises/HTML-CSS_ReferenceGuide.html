HTML & CSS Quick Reference Guide

HTML Tags

1. Paragraph (<p>)

Defines a paragraph of text.

<p>This is a paragraph.</p>

2. Heading (<h1> - <h6>)

Defines different levels of headings.

<h1>Main Heading</h1>
<h2>Subheading</h2>

3. Image (<img>)

Embeds an image into the page.

<img src="image.jpg" alt="Description">

4. Link (<a>)

Creates a hyperlink to another webpage.

<a href="https://example.com">Visit Example</a>

CSS Properties

1. Color

Sets the text color.

p {
    color: blue;
}

2. Font Size

Defines the size of the text.

h1 {
    font-size: 24px;
}
